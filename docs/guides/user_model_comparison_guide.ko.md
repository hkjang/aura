# 모델 비교 (Model Comparison) 가이드

이 문서는 Aura 포털의 **모델 비교** 기능을 사용하여 작업에 최적화된 AI 모델을 선택하는 방법을 설명합니다.

**메뉴 위치:** `AI 사용 > 모델 비교`

---

## 1. 개요

동일한 프롬프트(질문)를 여러 AI 모델에 동시에 전송하여 응답 속도, 품질, 비용 등을 비교분석할 수 있는 도구입니다.

### 주요 이점

- **최적의 모델 선정**: 특정 작업(창작, 번역, 논리 추론 등)에 가장 잘 맞는 모델을 찾을 수 있습니다.
- **성능 검증**: 신규 출시된 모델의 성능을 기존 모델과 직접 비교할 수 있습니다.
- **비용 관리**: 비슷한 품질을 제공하는 더 저렴한 모델을 식별하여 비용을 절감할 수 있습니다.

---

## 2. 사용 방법

### 2.1 비교 그룹 설정

1. 좌측의 **'비교 모델 선택'** 패널을 엽니다.
2. 비교하고 싶은 모델을 2개 이상 선택합니다. (최대 4개 권장)
   - 예: `GPT-4o` vs `Claude 3.5 Sonnet` vs `Llama 3`
3. 필요한 경우 각 모델의 파라미터(Temperature, Max Tokens)를 동일하게 맞춥니다.

### 2.2 테스트 프롬프트 입력

1. 중앙 입력창에 테스트할 프롬프트를 입력합니다.
2. **[비교 시작]** 버튼을 클릭합니다.

### 2.3 결과 분석

각 모델의 응답이 카드 형태로 나란히 표시됩니다.

- **응답 품질**: 생성된 텍스트의 정확성, 문체, 길이를 비교합니다.
- **지연 시간 (Latency)**: 첫 단어가 생성되기까지 걸린 시간(TTFT)과 전체 완료 시간을 확인합니다.
- **토큰 사용량**: 입력 및 출력 토큰 수를 비교하여 비용 효율성을 판단합니다.

---

## 3. 활용 시나리오

| 목적              | 추천 테스트                                 |
| :---------------- | :------------------------------------------ |
| **창의적 글쓰기** | "가을 아침을 주제로 짧은 시를 써줘"         |
| **논리적 추론**   | 복잡한 수학 문제나 논리 퀴즈 입력           |
| **정보 요약**     | 긴 뉴스 기사를 붙여넣고 "3줄 요약해줘" 요청 |
| **번역 품질**     | 전문 용어가 포함된 기술 문서를 번역 요청    |

---

## 4. 결과 저장

비교 결과가 마음에 드는 경우 **[리포트 저장]** 버튼을 눌러 비교 세션을 저장하거나, 특정 모델의 답변을 채택하여 이어서 대화를 진행할 수 있습니다.
