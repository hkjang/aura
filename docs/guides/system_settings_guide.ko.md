# 시스템 설정 (Settings) 가이드

이 문서는 Aura 포털의 동작 방식을 전역적으로 제어하는 **설정** 메뉴에 대한 가이드입니다.

**메뉴 위치:** `시스템 > 설정`

---

## 1. 일반 설정 (General)

### 브랜딩 (Branding)

- **사이트 이름**: 브라우저 탭에 표시될 이름을 변경합니다. (예: "OOO사 AI 포털")
- **로고 업로드**: 좌측 상단에 표시될 회사 로고 이미지를 업로드합니다.
- **테마 색상**: UI의 메인 컬러(Primary Color)를 지정합니다.

---

## 2. 모델 공급자 (Model Providers)

AI 모델을 제공하는 엔진과의 연결 정보를 설정합니다.

### OpenAI

- **API Key**: 발급받은 `sk-...` 키를 입력합니다.
- **Organization ID**: (선택) 조직 ID를 입력합니다.

### Ollama (로컬)

- **Base URL**: Ollama 서버 주소(예: `http://localhost:11434`)를 입력합니다.
- **사용 가능 모델**: 연결 테스트 후 감지된 모델 목록(llama3, mistral 등) 중 포털에서 노출할 모델을 체크합니다.

---

## 3. 보안 설정 (Security)

- **세션 만료 시간**: 로그인 후 아무런 활동이 없을 때 자동으로 로그아웃되는 시간을 설정합니다 (분 단위).
- **비밀번호 정책**: 비밀번호 최소 길이, 특수문자 포함 여부 등 복잡도 요구사항을 설정합니다.
