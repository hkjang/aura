# 오프라인 모드 (Offline Mode) 가이드

이 문서는 인터넷 연결이 차단된 폐쇄망 환경에서 Aura 포털을 운영하는 **오프라인 모드**에 대한 가이드입니다.

**메뉴 위치:** `시스템 > 오프라인`

---

## 1. 개요

보안을 위해 외부 인터넷과 단절된(Air-gapped) 환경에서도 AI 서비스를 원활하게 제공하기 위한 기능입니다.

---

## 2. 오프라인 기능

### 2.1 로컬 모델 서빙 (Local Model Serving)

- 인터넷이 필요한 OpenAI 대신, 사내 서버에 설치된 오픈소스 모델(Llama 3, Mistral 등)을 사용합니다.
- Ollama 또는 vLLM과 같은 로컬 추론 엔진과 연동됩니다.

### 2.2 데이터 동기화 관리

- 완전 폐쇄망이 아닌 경우(망분리 등), 주기적으로 외부의 업데이트(보안 패치, 모델 가중치)를 수동으로 가져오는 절차를 안내합니다.
- **[업데이트 패키지 업로드]** 기능을 통해 USB 등으로 반입된 업데이트 파일을 시스템에 적용할 수 있습니다.

---

## 3. 제약 사항

오프라인 모드에서는 외부 API를 호출하는 기능이 제한됩니다.

- **불가**: 웹 검색 플러그인, 클라우드 기반 LLM(GPT-4 등), 외부 이미지 생성 API
- **가능**: 로컬 LLM 채팅, 내부 문서 RAG, 내부 코드 생성

상세한 설치 및 배포 방법은 [오프라인 배포 가이드](../deployment/offline_deployment.ko.md)를 참조하세요.
