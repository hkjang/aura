# AI 사용 (AI Use) 가이드

이 문서는 Aura 포털의 핵심 기능인 "AI 사용" 메뉴의 하위 항목들에 대한 상세 가이드입니다.

## 1. 채팅 (Chat)

다양한 AI 모델과 실시간으로 대화할 수 있는 인터페이스입니다.

### 주요 기능

- **모델 선택**: 대화 상단에서 원하는 AI 모델(예: GPT-4, Claude 3, Llama 3)을 선택할 수 있습니다.
- **RAG 모드**: 업로드된 문서를 기반으로 답변하도록 설정할 수 있습니다.
- **히스토리**: 왼쪽 사이드바에서 과거 대화 내용을 확인하고 이어서 대화할 수 있습니다.
- **시스템 프롬프트**: AI의 역할이나 어조를 설정할 수 있는 시스템 메시지를 입력할 수 있습니다.

### 사용 팁

- 복잡한 질문은 단계별로 나누어 질문하면 더 정확한 답변을 얻을 수 있습니다.
- 특정 문서의 내용을 물어볼 때는 반드시 해당 문서가 지식 베이스에 인덱싱되어 있는지 확인하세요.

## 2. 문서 요약 (Document Summarization)

긴 문서를 빠르게 파악해야 할 때 유용한 기능입니다.

### 사용 방법

1. PDF, DOCX, 또는 TXT 파일을 업로드합니다.
2. "요약하기" 버튼을 클릭합니다.
3. AI가 생성한 핵심 요약, 주요 키워드, 그리고 세부 내용을 확인합니다.

## 3. 코드 생성 (Code Generation)

개발자를 위한 전문 코딩 보조 도구입니다.

### 기능

- **코드 작성**: 원하는 기능을 자연어로 설명하면 코드를 작성해줍니다.
- **디버깅**: 에러 메시지와 코드를 입력하면 원인과 해결책을 제시합니다.
- **리팩토링**: 기존 코드를 더 효율적이고 읽기 좋게 개선해줍니다.
- **언어 변환**: A 언어로 작성된 코드를 B 언어로 변환할 수 있습니다.

## 4. 모델 비교 (Model Comparison)

최적의 모델을 선정하기 위한 비교 도구입니다.

### 활용 시나리오

- **비용 최적화**: 성능 차이가 크지 않다면 더 저렴한 모델을 선택하기 위해 비교합니다.
- **성능 검증**: 특정 작업(예: 번역, 코딩)에 어떤 모델이 더 우수한지 직접 테스트합니다.

## 5. 에이전트 (Agents)

단순 대화를 넘어 작업을 수행하는 자율 에이전트입니다.

### 에이전트 유형

- **조사 에이전트**: 웹 검색을 통해 특정 주제를 심층 조사하고 보고서를 작성합니다.
- **코딩 에이전트**: 복잡한 기능을 구현하고 테스트 케이스까지 작성합니다.
- **워크플로우 에이전트**: 사전에 정의된 업무 프로세스(승인 요청 -> 메일 발송 등)를 자동화합니다.

## 6. 프롬프트 (Prompts)

자주 사용하는 명령어를 템플릿으로 저장하고 관리합니다.

### 기능

- **템플릿 생성**: 자주 쓰는 프롬프트를 변수(`{{variable}}`)와 함께 저장합니다.
- **공유**: 유용한 프롬프트를 팀원들과 공유할 수 있습니다.
- **버전 관리**: 프롬프트의 이력을 관리하고 개선할 수 있습니다.
