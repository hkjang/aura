# 사용자 관리 (User Management) 가이드

이 문서는 시스템 접근 권한을 제어하는 **사용자 관리** 기능에 대한 가이드입니다.

**메뉴 위치:** `시스템 > 사용자`

---

## 1. 개요

관리자는 이 메뉴에서 새로운 사용자를 추가하거나, 기존 사용자의 권한을 수정하고, 퇴사자의 계정을 비활성화할 수 있습니다.

---

## 2. 사용자 목록 및 검색

기본 화면에서 등록된 모든 사용자의 이름, 이메일, 역할, 가입일, 상태(활성/비활성)를 목록으로 확인할 수 있습니다. 상단 검색창을 통해 특정 사용자를 빠르게 찾을 수 있습니다.

---

## 3. 주요 작업

### 3.1 사용자 추가

1. **[사용자 추가]** 버튼을 클릭합니다.
2. 필수 정보(이메일, 이름, 비밀번호)를 입력합니다.
3. 초기 역할을 지정합니다.
4. 생성이 완료되면 해당 사용자에게 이메일이나 사내 메신저로 접속 정보를 전달합니다.

### 3.2 역할(Role) 수정

사용자의 상세 페이지에서 역할을 변경할 수 있습니다.

- **USER**: 일반 사용자. 채팅, 검색 등 기본 기능만 사용 가능.
- **ADMIN**: 시스템 전체 설정, 사용자 관리, 감사 로그 열람 가능.

### 3.3 계정 비활성화/삭제

- **비활성화**: 사용자의 로그인을 일시적으로 막습니다. (휴직, 감사 중 등)
- **삭제**: 계정 정보를 영구히 삭제합니다. 단, 감사 로그 보존을 위해 삭제보다는 비활성화를 권장합니다.
