# 시스템 (System) 가이드

시스템 관리자를 위한 설정, 사용자 관리, 로그 감사 기능을 설명합니다.

## 1. 시스템 상태 (SRE)

IT 운영팀을 위한 실시간 인프라 모니터링 도구입니다.

### 모니터링 항목

- **API 지연 시간(Latency)**: AI 모델의 응답 속도 추이.
- **시스템 리소스**: 서버의 CPU, 메모리, GPU 사용량 (자체 호스팅 시).
- **외부 서비스 상태**: OpenAI, AWS 등 외부 API의 장애 여부.

## 2. 오프라인 모드 (Offline)

폐쇄망 환경 운영을 위한 설정입니다.

### 설정 항목

- **인터넷 차단**: 외부 인터넷 연결 시도를 강제로 차단합니다.
- **로컬 모델 엔드포인트**: 내부망에 설치된 Ollama 또는 vLLM 서버 주소를 지정합니다.
- **업데이트 관리**: 오프라인 환경에서의 수동 업데이트 절차를 지원합니다.

## 3. 사용자 관리 (Users)

포털 사용자 계정 및 권한을 관리합니다.

### 기능

- **사용자 추가/초대**: 이메일 초대를 보내거나 관리자가 직접 계정을 생성합니다.
- **역할 할당**:
  - **USER**: 일반 사용자. AI 기능 사용만 가능.
  - **ADMIN**: 전체 관리자. 모든 설정 변경 가능.
  - **AUDITOR**: 감사 로그 열람만 가능한 역할 (설정 시).
- **계정 비활성화**: 퇴사자 등의 계정을 즉시 정지시킵니다.

## 4. 설정 (Settings)

시스템 전반에 적용되는 글로벌 설정을 변경합니다.

- **브랜딩**: 로고, 회사명, UI 테마 색상을 변경합니다.
- **기본 모델**: 채팅 시작 시 기본적으로 선택될 모델을 지정합니다.
- **공지사항**: 대시보드 상단에 전사 공지사항을 배너로 띄웁니다.

## 5. 로그 (Logs) & 감사 (Audit)

보안 감사를 위한 기록 보관소입니다.

- **시스템 로그**: 에러, 경고 등 시스템 동작 관련 기술적 로그.
- **감사 로그(Audit Trail)**:
  - 누가 언제 어떤 설정을 변경했는지.
  - 누가 어떤 문서를 다운로드했는지.
  - 권한 변경 이력 등 중요한 관리 작업의 기록.
