# 문서 요약 (Document Summarization) 가이드

이 문서는 Aura 포털의 **문서 요약** 기능을 사용하여 긴 문서를 효율적으로 분석하는 방법을 설명합니다.

**메뉴 위치:** `AI 사용 > 문서 요약`

---

## 1. 개요

문서 요약 기능은 긴 보고서, 논문, 회의록 등을 AI를 통해 빠르게 분석하여 핵심 내용, 키워드, 주요 논점을 추출해주는 도구입니다.

### 지원 파일 형식

- **PDF (.pdf)**: 텍스트 추출이 가능한 전자 문서. (이미지 위주의 스캔 문서는 지원되지 않을 수 있음)
- **Word (.docx)**: 표준 Microsoft Word 문서.
- **Text (.txt)**: 일반 텍스트 파일.

---

## 2. 사용 방법

### 1단계: 문서 업로드

1. 파일 업로드 영역을 클릭하거나 파일을 드래그하여 놓습니다.
2. 업로드된 파일명과 크기가 표시되는지 확인합니다.
3. _팁: 텍스트 양이 매우 많은 문서는 처리 시간이 길어질 수 있습니다._

### 2단계: 요약 옵션 선택

문서의 성격과 필요한 정보량에 따라 요약 길이를 선택합니다.

- **간단 (Short)**: 1-2 문장으로 내용을 초압축합니다. 제목이나 헤드라인 수준의 요약이 필요할 때 사용합니다.
- **중간 (Medium)**: 3-5 문장으로 핵심 내용을 전달합니다. 일반적인 개요 파악에 적합합니다. (기본값)
- **상세 (Detailed)**: 전체 내용을 포괄하는 상세한 요약을 제공합니다. 보고서 작성 시 참고 자료로 활용하기 좋습니다.

### 3단계: 요약 생성

**[AI 요약 생성]** 버튼을 클릭하면 분석이 시작됩니다. 문서 길이에 따라 수 초에서 수십 초가 소요될 수 있습니다.

---

## 3. 요약 결과 활용

분석이 완료되면 다음과 같은 정보가 제공됩니다.

### 결과 항목

1. **요약 (Summary)**: AI가 작성한 줄글 형태의 요약문입니다.
2. **핵심 포인트 (Key Points)**: 주요 논점이나 결정 사항을 번호 매기기 목록으로 정리한 것입니다.
3. **키워드 (Keywords)**: 문서에서 가장 자주 등장하거나 중요한 단어 태그입니다.
4. **문서 통계**: 원본 단어 수와 예상 읽기 시간을 제공하여 업무량 예측을 돕습니다.

### 저장 및 공유

결과 카드의 우측 상단 버튼을 활용하세요.

- **복사 (Copy)**: 요약 결과를 클립보드에 복사하여 이메일이나 메신저에 붙여넣을 수 있습니다.
- **다운로드 (Download)**: 요약 내용, 핵심 포인트, 키워드, 통계 정보를 포함한 `.txt` 파일로 저장합니다.

## 4. 자주 묻는 질문 (FAQ)

**Q: 스캔한 PDF도 요약되나요?**
A: 현재 버전에서는 이미지로 된 텍스트(OCR이 필요한 문서)는 정확도가 낮을 수 있습니다. 텍스트 드래그가 가능한 PDF 사용을 권장합니다.

**Q: 파일 크기 제한이 있나요?**
A: 서버 설정에 따라 다르지만 기본적으로 10MB 미만의 파일을 권장합니다.

**Q: 요약 내용이 원문과 다른 것 같습니다.**
A: AI는 문맥을 재구성하여 요약하므로 표현이 달라질 수 있습니다. 중요한 의사결정 시에는 반드시 원문의 해당 부분을 다시 확인하시기 바랍니다.
