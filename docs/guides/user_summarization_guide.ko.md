# 문서 요약 (Document Summarization) 가이드

이 문서는 Aura 포털의 **문서 요약** 기능을 사용하여 긴 문서를 효율적으로 분석하는 방법을 설명합니다.

**메뉴 위치:** `AI 사용 > 문서 요약`

---

## 1. 개요

문서 요약 기능은 긴 보고서, 논문, 회의록 등을 AI를 통해 빠르게 분석하여 핵심 내용, 키워드, 주요 논점을 추출해주는 도구입니다.

### 지원 파일 형식

- **PDF (.pdf)**: 텍스트 추출이 가능한 전자 문서. (이미지 위주의 스캔 문서는 지원되지 않을 수 있음)
- **Word (.docx)**: 표준 Microsoft Word 문서.
- **Text (.txt)**: 일반 텍스트 파일.
- **Markdown (.md)**: 개발 문서 또는 기술 블로그 등.

---

## 2. 사용 방법

### 2.1 파일 업로드 및 설정

1. **업로드**: 파일 업로드 영역을 클릭하거나 파일을 드래그하여 놓습니다. (최대 10MB)
2. **언어 감지**: 문서는 자동으로 언어가 감지되지만, 요약 결과의 언어를 강제로 지정할 수도 있습니다 (예: 영어 논문을 한국어로 요약).

### 2.2 요약 옵션 (Summary Options)

문서의 성격과 필요한 정보량에 따라 요약 모드를 선택하세요.

| 옵션                      | 설명                                  | 추천 대상                 |
| :------------------------ | :------------------------------------ | :------------------------ |
| **개요 (Brief)**          | 1-2 문장으로 전체 주제만 빠르게 파악  | 뉴스 기사, 이메일         |
| **표준 (Standard)**       | 3-5 문단으로 핵심 내용 요약 (기본값)  | 일반 보고서, 블로그       |
| **상세 (Detailed)**       | 챕터별/섹션별 상세 요약 제공          | 논문, 계약서, 기술 명세서 |
| **불렛 포인트 (Bullets)** | 줄글 대신 핵심 사항을 개조식으로 나열 | 회의록, 투두 리스트       |

### 2.3 고급 설정 (Advanced)

- **모델 선택**: 기본적으로 빠르고 저렴한 모델(GPT-3.5 등)이 사용되지만, 복잡한 문서의 경우 `GPT-4`나 `Claude 3 Opus`를 선택하여 정확도를 높일 수 있습니다.

---

## 3. 요약 결과 활용

분석이 완료되면 우측 패널에 결과가 표시됩니다.

### 결과 항목

1. **Executive Summary**: 경영진 보고용 초고속 요약.
2. **Key Takeaways**: 문서에서 얻어가야 할 핵심 통찰 3~5가지.
3. **Keywords**: 문서 내 빈출 중요 단어 태그.
4. **Action Items**: 문서 내용 중 '할 일'이나 '결정 사항'이 포함된 경우 이를 별도로 추출합니다.

### 저장 및 공유

- **노트북으로 내보내기**: 요약 결과를 내 개인 노트북에 바로 저장하여 나중에 RAG(검색)에 활용할 수 있습니다.
- **다운로드**: 원본 텍스트와 요약본을 `.txt` 또는 `.pdf`로 내려받습니다.

---

## 4. 문제 해결 (Troubleshooting)

**Q: "텍스트를 추출할 수 없습니다"라는 오류가 뜹니다.**
A: 암호가 걸린 PDF이거나, 텍스트가 아닌 이미지로만 구성된 스캔 파일일 수 있습니다. OCR(광학 문자 인식) 처리가 된 파일을 사용해주세요.

**Q: 요약하다가 중간에 멈춥니다.**
A: 문서가 매우 긴 경우(100페이지 이상), AI 모델의 처리 한도(Context Window)를 초과했을 수 있습니다. 문서를 분할하여 업로드하거나, 토큰 처리량이 큰 모델(예: Claude 3 200k)을 선택하세요.

**Q: 전문 용어를 잘 못 알아듣습니다.**
A: 시스템 설정의 '지식 베이스'에 사내 용어집을 등록해두면 정확도가 향상될 수 있습니다. (관리자 문의 필요)
