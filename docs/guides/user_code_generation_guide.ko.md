# 코드 생성 (Code Generation) 가이드

이 문서는 Aura 포털의 **코드 생성** 기능을 활용하여 개발 생산성을 높이는 방법을 설명합니다.

**메뉴 위치:** `AI 사용 > 코드 생성`

---

## 1. 개요

코드 생성 기능은 전문 개발자와 초보자 모두를 위해 설계된 강력한 코딩 보조 도구입니다. 자연어 설명을 코드로 변환하거나, 기존 코드를 분석하여 개선할 수 있습니다.

### 주요 기능

- **코드 작성 (Generation)**: 요구사항을 입력하면 실행 가능한 코드를 생성합니다.
- **디버깅 (Debugging)**: 에러가 발생한 코드를 수정하고 원인을 설명합니다.
- **리팩토링 (Refactoring)**: 코드 구조를 개선하여 가독성과 성능을 높입니다.
- **주석 추가 (Documentation)**: 복잡한 코드에 자동으로 주석을 달아줍니다.

---

## 2. 사용 방법

### 2.1 코드 생성하기

1. **언어 선택**: Python, JavaScript, Java 등 원하는 프로그래밍 언어를 선택합니다.
2. **요구사항 입력**:
   - 예: "React로 드래그 앤 드롭이 가능한 할 일 목록 컴포넌트를 만들어줘."
   - _팁: 입력/출력 예시를 포함하면 더 정확한 결과를 얻을 수 있습니다._
3. **생성**: AI가 코드를 작성하며, 필요한 라이브러리 설치 방법도 함께 제공합니다.

### 2.2 기존 코드 수정 (디버깅/리팩토링)

1. **코드 붙여넣기**: 에디터 영역에 기존 코드를 입력합니다.
2. **모드 선택**:
   - `Debug`: 버그 수정 및 에러 해결
   - `Refactor`: 코드 구조 개선
   - `Optimize`: 성능 최적화
3. **실행**: AI가 수정된 코드를 제안하고 변경된 이유를 설명합니다.

---

## 3. 고급 활용 팁

### 컨텍스트 유지

이전 대화 내용을 기억하므로, "이 코드에 에러 처리를 추가해줘" 또는 "변수명을 스네이크 케이스로 바꿔줘"와 같이 연속적인 명령이 가능합니다.

### 프롬프트 예시

- **단위 테스트 생성**: "위 함수에 대한 Jest 단위 테스트 코드를 작성해줘."
- **API 연동**: "Axios를 사용하여 날씨 API를 호출하고 결과를 화면에 표시하는 함수를 만들어줘."
- **SQL 쿼리**: "사용자 테이블에서 최근 30일간 로그인하지 않은 유저를 찾는 SQL 쿼리를 작성해줘."

---

## 4. 주의사항

- 생성된 코드는 반드시 테스트 환경에서 검증 후 프로덕션에 적용하세요.
- 보안에 민감한 정보(API 키, 비밀번호 등)가 포함된 코드는 직접 입력하지 마세요.
