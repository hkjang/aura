# MLOps 가이드

이 문서는 AI 모델의 운영 수명 주기를 관리하는 **MLOps** 기능에 대한 가이드입니다.

**메뉴 위치:** `분석 > MLOps`

---

## 1. 개요

MLOps(Machine Learning Operations)는 모델의 배포, 모니터링, 재학습 등 운영 전반을 관리하는 체계입니다. Aura 포털에서는 특히 **LLM의 성능 평가 및 배포**에 초점을 맞춥니다.

---

## 2. 모델 평가 (Model Evaluation)

새로운 모델이나 프롬프트를 프로덕션에 적용하기 전에 테스트하는 기능입니다.

### 골든 셋 (Golden Set) 테스트

1. **데이터셋 준비**: 정답이 있는 질문-답변 세트(Golden Set)를 준비합니다.
2. **자동 평가**: 여러 모델에게 동일한 질문을 던지고, '정확도', '유사도' 등의 지표로 점수를 매깁니다.
3. **비교**: 기존 모델 대비 신규 모델의 점수가 높은지 확인합니다.

---

## 3. 파인튜닝 (Fine-tuning) 현황

(일부 모델 지원 시) 사내 데이터를 학습시킨 커스텀 모델의 학습 진행 상황과 성능 지표(Loss Curve 등)를 모니터링할 수 있습니다.
