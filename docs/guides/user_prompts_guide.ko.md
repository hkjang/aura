# 프롬프트 (Prompts) 가이드

이 문서는 고품질의 AI 응답을 얻기 위해 **프롬프트 템플릿**을 생성, 관리, 공유하는 방법을 설명합니다.

**메뉴 위치:** `AI 사용 > 프롬프트`

---

## 1. 개요

매번 동일한 지시사항을 반복해서 입력할 필요 없이, 미리 정의된 템플릿을 사용하여 업무 효율을 높이는 기능입니다. 팀원들과 유용한 프롬프트를 공유하여 조직 전체의 AI 활용 역량을 상향 평준화할 수 있습니다.

---

## 2. 프롬프트 관리

### 2.1 프롬프트 생성

1. **[새 프롬프트]** 버튼을 클릭합니다.
2. **제목**과 **설명**을 입력합니다.
3. **템플릿 내용**을 작성합니다. 변수가 필요한 부분은 중괄호(`{{ }}`)를 사용합니다.

   **예시 (이메일 작성 템플릿):**

   ```text
   당신은 전문 비즈니스 이메일 작성자입니다.

   수신자: {{recipient}}
   목적: {{purpose}}
   어조: {{tone}} (예: 정중하게, 친근하게)
   핵심 내용: {{key_points}}

   위 정보를 바탕으로 비즈니스 이메일 초안을 작성해주세요.
   ```

### 2.2 변수(Variables) 활용

템플릿 내에 `{{변수명}}`을 사용하면, 채팅창에서 해당 프롬프트를 불러올 때 입력 폼이 자동으로 생성됩니다. 사용자는 빈칸만 채우면 완성된 프롬프트가 AI에게 전송됩니다.

### 2.3 태그 및 카테고리

검색이 쉽도록 `#마케팅`, `#개발`, `#보고서` 등의 태그를 추가하여 관리하세요.

---

## 3. 프롬프트 사용하기

### 채팅에서 불러오기

1. 채팅 입력창에서 슬래시(`/`) 키를 입력합니다.
2. 프롬프트 목록이 팝업되면 원하는 템플릿을 선택하거나 제목을 검색합니다.
3. 변수 입력 폼이 뜨면 필요한 내용을 입력하고 전송합니다.

---

## 4. 프롬프트 최적화 팁

- **명확한 지시**: AI에게 원하는 역할("You are a expert...")을 부여하세요.
- **예시 제공 (Few-shot)**: 원하는 결과물의 예시를 1~2개 포함하면 품질이 대폭 향상됩니다.
- **단계별 사고 (Chain of Thought)**: "단계별로 생각해서 답해줘"라는 문구를 추가하면 논리적인 답변을 얻을 수 있습니다.
