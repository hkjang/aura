# 문서 관리 (Documents) 가이드

이 문서는 지식 베이스(RAG용)와 구분되는, Aura 포털 내의 일반 **문서 자산 관리** 기능을 설명합니다.

**메뉴 위치:** `관리 > 문서`

---

## 1. 개요

지식 베이스가 "AI를 위한 문서"라면, 이 메뉴는 "사람을 위한 문서"를 관리하는 공간입니다. 포털 사용자들에게 공유해야 할 가이드, 템플릿, 공지 파일 등을 중앙에서 관리합니다.

### 활용 예시

- **사용자 매뉴얼 배포**: Aura 포털 사용법 가이드 문서 공유
- **프롬프트 엔지니어링 팁**: 전사적으로 공유할 팁 모음집
- **정책 문서**: AI 사용 윤리 규정 파일 등

---

## 2. 기능 설명

### 2.1 파일 업로드 및 버전 관리

- 파일을 업로드하고 설명을 덧붙일 수 있습니다.
- 동일한 파일명으로 새 버전을 업로드하면 버전 이력이 관리됩니다.

### 2.2 권한 설정 (Permissions)

모든 사용자가 볼 수 있게 할지, 특정 그룹(예: 관리자)만 볼 수 있게 할지 설정합니다.

- **Public**: 로그인한 모든 사용자 열람 가능
- **Admin Only**: 관리자 권한 사용자만 열람 가능
- **Private**: 업로더 본인만 확인 가능 (초안 작성용)

### 2.3 미리보기 및 다운로드

- 웹 브라우저에서 바로 문서를 미리보기 할 수 있습니다. (PDF, 이미지 등 지원)
- 원본 파일을 다운로드할 수 있습니다.

---

## 3. 지식 베이스와의 차이점

| 구분          | 문서 (Documents)              | 지식 베이스 (Knowledge Base) |
| :------------ | :---------------------------- | :--------------------------- |
| **주 목적**   | 사람이 읽고 다운로드하기 위함 | AI가 검색하고 답변하기 위함  |
| **처리 방식** | 원본 파일 그대로 저장         | 벡터로 변환(임베딩)하여 저장 |
| **AI 검색**   | 불가능 (단순 저장소)          | 가능 (RAG 소스)              |
