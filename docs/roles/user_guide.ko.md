# 사용자 (USER) 가이드

이 문서는 Aura 포털의 일반 사용자(USER)를 위한 상세 가이드입니다.

## 1. 시작하기

### 로그인

관리자로부터 받은 계정 정보나 사내 이메일 주소를 사용하여 로그인합니다.

### 대시보드 개요

로그인 후 처음 접하는 화면입니다.

- **최근 대화**: 이전에 나눴던 채팅 기록에 바로 접근할 수 있습니다.
- **빠른 실행**: 문서 요약, 코드 생성 등 자주 쓰는 기능의 바로가기 버튼이 있습니다.

## 2. 핵심 업무 시나리오

### 시나리오 A: 보고서 작성을 위한 정보 검색

1. **[채팅]** 메뉴로 이동합니다.
2. "인터넷 검색" 플러그인이 켜져 있는지 확인합니다 (가능한 경우).
3. "최근 AI 트렌드에 대해 조사해줘"라고 입력합니다.
4. AI가 제공한 정보를 바탕으로 추가 질문을 이어갑니다.
5. "이 내용을 바탕으로 보고서 개요를 작성해줘"라고 요청합니다.

### 시나리오 B: 사내 규정 확인하기 (RAG)

1. **[채팅]** 메뉴로 이동합니다.
2. 모델 선택기 옆의 **[지식 베이스]** 드롭다운에서 "사내 규정" 컬렉션을 선택합니다.
3. "재택근무 신청 절차가 어떻게 돼?"라고 물어봅니다.
4. AI가 관련 문서를 참조하여 답변과 함께 출처를 표시해줍니다.

### 시나리오 C: 회의록 요약

1. **[문서 요약]** 메뉴로 이동합니다.
2. 회의록 파일(PDF, Word)을 업로드합니다.
3. 요약 옵션에서 "핵심 안건 및 결정 사항 위주"를 선택합니다.
4. 생성된 요약을 다운로드하거나 복사합니다.

## 3. 프롬프트 활용하기

반복적인 업무는 프롬프트 템플릿을 만들어 효율화할 수 있습니다.

### 나만의 템플릿 만들기

1. **[프롬프트]** 메뉴로 이동합니다.
2. **[새 프롬프트]** 버튼을 클릭합니다.
3. 내용에 변수를 사용하여 템플릿을 만듭니다.
   - 예: `다음 [{{topic}}]에 대한 마케팅 문구를 3가지 버전(전문적, 친근함, 위트있음)으로 작성해줘.`
4. 저장 후 채팅 화면에서 `/`를 입력하여 바로 호출할 수 있습니다.

## 4. 자주 묻는 질문 (FAQ)

**Q: 답변이 너무 느립니다.**
A: 복잡한 추론이 필요한 모델(예: GPT-4)을 사용 중일 수 있습니다. 간단한 질문은 더 빠른 모델(예: GPT-3.5, Llama 3)로 변경해보세요.

**Q: "보안 정책에 의해 차단되었습니다"라고 뜹니다.**
A: 입력한 내용이나 AI의 답변에 사내 보안 규정(PII, 금지어)에 위배되는 내용이 포함되어 있을 수 있습니다.

**Q: 제가 올린 문서를 다른 사람이 볼 수 있나요?**
A: 기본적으로 본인이 업로드한 문서는 비공개입니다. 관리자에게 요청하여 공용 지식 베이스에 추가할 수 있습니다.
