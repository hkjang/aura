# Aura 엔터프라이즈 AI 포털 - 사용자 매뉴얼

## 1. 소개

Aura는 대규모 언어 모델(LLM)에 접근하기 위한 안전하고 통일된 인터페이스를 제공하는 엔터프라이즈급 AI 포털입니다. 강력한 보안 정책에 따라 채팅, 문서 분석, 코드 생성 및 자율 에이전트 기능을 통합하여 제공합니다.

---

## 2. 사용자 역할

### 👤 USER (일반 사용자)

- **접근 권한**: "AI 사용" 메뉴의 표준 AI 기능에 접근할 수 있습니다.
- **기능**: AI와 채팅, 문서 요약, 코드 생성, 개인 로그 확인 등이 가능합니다.
- **제한 사항**: 시스템 설정을 수정하거나, 타 사용자를 관리하거나, 거버넌스 규칙을 설정할 수 없습니다.

### 🛡️ ADMIN (관리자)

- **접근 권한**: "관리", "분석", "시스템"을 포함한 모든 메뉴에 접근할 수 있습니다.
- **기능**:
  - 사용자 및 역할 관리.
  - AI 모델 및 키 구성.
  - 거버넌스 정책 설정 (PII 필터링, 금지 주제).
  - 비용, 품질 및 시스템 상태 모니터링.
  - 지식 베이스 및 RAG 설정 관리.

---

## 3. 메뉴 가이드

사이드바는 목적에 따라 4가지 주요 섹션으로 구분됩니다.

### 🤖 AI 사용 (AI Use)

_모든 사용자가 접근 가능합니다._

- **채팅 (Chat)**: AI 모델과 대화하기 위한 기본 인터페이스입니다. 문맥 설정 및 모델 선택을 지원합니다.
- **문서 요약 (Document Summarization)**: PDF나 텍스트 문서를 업로드하여 AI가 생성한 간결한 요약을 받습니다.
- **코드 생성 (Code Generation)**: 코드 조각 생성, 디버깅 및 리팩토링을 위한 특수 인터페이스입니다.
- **모델 비교 (Model Comparison)**: 작업에 가장 적합한 모델을 선택하기 위해 서로 다른 AI 모델의 응답을 나란히 비교합니다.
- **에이전트 (Agents)**: 자율 AI 에이전트에게 복잡한 다단계 작업을 할당합니다.
- **프롬프트 (Prompts)**: 일관된 결과를 위해 저장된 프롬프트 템플릿을 관리하고 사용합니다.

### 🛠 관리 (Manage)

_주로 관리자와 지식 관리자를 위한 메뉴입니다._

- **지식 베이스 (Knowledge Base)**: RAG(검색 증강 생성) 인덱스를 관리합니다. AI가 검색할 수 있도록 문서를 업로드하고 인덱싱합니다.
- **문서 관리 (Documents)**: 업로드된 자산에 대한 일반적인 파일 관리입니다.
- **플러그인 (Plugins)**: AI가 사용할 수 있는 외부 통합 및 도구를 관리합니다.
- **거버넌스 (Governance)**: 보안 정책을 구성합니다.
  - **정책 (Policies)**: PII 필터, 차단된 키워드 및 주제 금지를 정의합니다.
  - **감사 (Audit)**: 정책 위반 로그를 봅니다.

### 📊 분석 (Analyze)

_관리자와 분석가를 위한 메뉴입니다._

- **대시보드 (Dashboard)**: 시스템 사용량 및 주요 지표에 대한 고수준 개요입니다.
- **품질 분석 (Quality)**: 사용자 피드백 및 자동 점수 매기기를 기반으로 AI 응답의 정확성과 관련성을 분석합니다.
- **비용 분석 (Cost)**: 모델, 사용자 부서별 토큰 사용량 및 예상 비용을 추적합니다.
- **ML 운영 (MLOps)**: 모델 성능, 배포 상태 및 버전 관리를 모니터링합니다.

### ⚙️ 시스템 (System)

_관리자 전용입니다._

- **시스템 상태 (SRE)**: 시스템 상태, API 상태 및 오류율을 실시간으로 모니터링합니다.
- **오프라인 모드 (Offline)**: 폐쇄망(에어갭) 환경을 위한 구성입니다.
- **사용자 관리 (Users)**: 역할 할당 및 권한 취소를 포함한 사용자 관리 기능입니다.
- **설정 (Settings)**: 전역 시스템 구성 (브랜딩, 기본 모델 등).
- **로그 (Logs)**: 문제 해결을 위한 상세 시스템 로그입니다.
- **감사 (Audit)**: 중요 작업(예: 역할 변경, 정책 업데이트)에 대한 포괄적인 감사 추적입니다.

## 4. 주요 기능 및 워크플로우

### RAG (지식 베이스) 사용 방법

1. **관리 > 지식 베이스**로 이동합니다.
2. PDF 또는 텍스트 문서를 업로드합니다.
3. 인덱싱 프로세스가 완료될 때까지 기다립니다.
4. **AI 사용 > 채팅**으로 이동합니다.
5. "RAG 모드"를 활성화하거나 업로드된 컬렉션을 선택합니다.
6. 질문을 하면 AI가 문서를 기반으로 답변합니다.

### 모델 비교 방법

1. **AI 사용 > 모델 비교**로 이동합니다.
2. 모델 A(예: GPT-4)와 모델 B(예: Llama 3)를 선택합니다.
3. 프롬프트를 입력합니다.
4. 품질과 속도를 평가하기 위해 두 응답을 나란히 확인합니다.
